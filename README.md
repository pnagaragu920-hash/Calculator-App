
# Calculator-App

## 🚀 Description
A simple calculator built using HTML, CSS, and JavaScript.

## 🔧 Features
- Addition, Subtraction, Multiplication, Division
- Clear button
- Dynamic display

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

## 📌 How to Use
1. Enter numbers using buttons
2. Click operator
3. Press "=" to get result
